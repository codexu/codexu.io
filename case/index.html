      # 征集更多案例

      如果你使用 X-BUILD 开发了某个项目，可以通过下面的方式进行分享：

      • 加 QQ群 或 作者微信(xu461229187)
      • 通过进入案例页面点击下方修改此页自行编辑和提交

      准备信息：

      • 【项目名称】：可附带项目源码链接
      • 【作者】：可附带个人 Github 链接
      • 【线上链接】：PC 采用链接形式，移动端请生成二维码(80*80)
      • 【备注信息】

      二维码路径 > /docs/.vuepress/public/qr/***.jpg
