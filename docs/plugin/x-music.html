X-MUSIC

X-MUSIC

设置背景音乐及播放控制按钮。

安装

  • 通过 npm or yarn 安装：
**[terminal]
**[command npm install x-music --save]

起步

  • 使用 ES6 方式引用：
import XMusic from 'x-music';

// 使用 webpack 请 import 引入静态资源
import src from '***';
import playImg from '***';
import pauseImg from '***';

const xMusic = new XMusic({ src, playImg, pauseImg });
xMusic.init();

本项目通过 rollup 使用 UMD 格式打包，支持 AMD、CommonJS、script 标签引入，下载源码

参数

如果使用 webpack 进行打包，请使用 import 加载静态资源：

src*

none <String>

音乐路径

playImg*

none <String>

播放按钮图片路径

pauseImg*

none <String>

暂停按钮图片路径

width

none <String>

按钮图片宽度(px)

height

none <String>

按钮图片高度(px)
