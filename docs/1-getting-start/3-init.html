创建一个项目

创建一个新项目

通过以下指令创建一个新项目：

  $ x-build create [name]

参数

参数 说明
-n, noversion ⚠️ 忽略版本检测，可能会影响项目正常启动。
-q, quick 快速初始化一个项目（无ESLint、CSS预处理器、REM布局）。
