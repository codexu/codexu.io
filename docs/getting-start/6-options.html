配置

配置

使用根目录下的 xbuild.config.js 文件，可以对项目做更多的个性会配置。

port

8080 <Number>

端口：改变 devServer 端口号，默认为 8080 端口。

open

false <boolean>

浏览器： 自动使用默认浏览器运行项目

entry

<Array> 
[
  './src/scripts/index.js',
  './src/styles/index.[css/scss/less/styl]'
]

入口文件： 默认设置了 index.js 和 index.[css/scss/less/styl] 的入口文件。

eslint

<boolean>

代码检测：如果初始化时选择了 true，会开启代码检测，可设置为 false 关闭 ESLint。

babel

<boolean>

代码编译：Babel 可以使用大部分浏览器未支持的语法。

mobileLayout

<boolean>

自适应布局：是否使用自适应布局。

designWidth

750<Number>

设计稿尺寸(px)

base64

8 * 1024<Number>

静态资源：设定静态资源使用 base64 打包的大小，默认为 8kb。

proxy

<Object> 
{
  '/api': {
    target: '#',
    pathRewrite: {
      '^/api': ''
    },
    changeOrigin: true,
    secure: false
  }
}

跨域：当遇到跨域问题时，可以通过此项配置，配置方法参考webpack-proxy。
