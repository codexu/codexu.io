# 创建一个新项目

![](https://ws3.sinaimg.cn/large/006tNbRwly1fwi9fxsva5g30qp0g1q5v.gif)

通过以下任意指令创建一个新项目：

```
  $ x-build init
  or
  $ x -i
```

&emsp;

你会被提示输入或选择一系列问题来自定义你的项目：

- 项目名称 *（默认x-build）*

- 初始版本 *（默认0.0.1）*

- server端口 *（默认8080）*

- 移动端开发？ *（默认yes）*

- 选择包管理器 *（yarn/npm/cnpm）*

- 选择安装插件 *（normalize.css/x-load/x-animate/animate.css）*

&emsp;

随后在github下载x-build最新版本 -> 安装项目依赖 -> 安装插件
