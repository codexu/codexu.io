什么是Pug

Pug是一种用于编写HTML的干净，空格敏感的语法，通过缩进的方式进行书写，例如：

#container.col
  p You are using pug!

编译为:

<div id="container" class="col">
  <p>You are using pug!</p>
</div>

可见，这样的写法可以省去大量的代码，但要注意缩进关系。
