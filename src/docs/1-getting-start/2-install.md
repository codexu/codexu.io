### 安装命令行工具 (CLI)

全局安装npm包，提供终端命令`x-build` or `x`。

使用下列任一命令安装：

```
  $ npm install -g x-build-cli
  $ yarn global add x-build-cli
```

- 使用`-V`查看版本号

- 使用`-n`参数禁止版本检测

- 使用`-h`查看参数列表
