<p align="center">X-BUILD 是面向小型项目的脚手架工具，通过终端命令最快在几秒钟初始化项目目录，解决开发中的自动化、性能、模块化、规范、自适应布局等问题，辅助你生成更优质的代码。</p>

![](http://ww4.sinaimg.cn/large/006tNc79ly1g3yi5adyjlg30rs0hzdqw.gif)

## 主要功能

- 快速生成工程目录

- 基于 webpack 的开发环境

- 自适应解决方案

- 数据接口模块自动加载

- CSS预处理器(sass、less、stylus)

- ESLint

- Babel
